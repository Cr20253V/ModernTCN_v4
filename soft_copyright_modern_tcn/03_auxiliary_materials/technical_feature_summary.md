# 技术特点摘要

## 1. 主要技术特点

1. **固定长度多变量时序窗口**：使用 128 步 × 19 维的滑动窗口组织输入数据，覆盖约 1.28 秒的历史观测，捕获 AGV 工况的时序相关性。

2. **19 维 AGV 观测特征**：输入特征涵盖纵向加速度、偏航角速度、驱动电流、轮速、转向角、俯仰角速度、估计速度、曲率代理量等 AGV 关键可观测量。

3. **三任务输出**：单个 ModernTCN 模型同时输出主工况分类（flat/stall/slope）、转向方向分类（right/straight/left）和坡度回归量（theta_hat），共享卷积骨干特征。

4. **大核深度时序卷积**：采用 kernel_size=31 的深度可分离卷积，配合残差连接和 layer_scale，在参数量可控的同时捕获长时间依赖。

5. **多 seed 训练与指标汇总**：支持多随机种子训练，汇总各 seed 的验证和测试指标，评估模型稳定性和泛化能力。

6. **ONNX 导出**：将 PyTorch 训练模型导出为 ONNX 格式，固定输入形状 [1, 128, 19]，使用标准算子确保跨平台兼容性。

7. **PyTorch/ONNXRuntime/MATLAB 一致性验证**：提供 Python 端和 MATLAB 端的一致性检查脚本，确保 ONNX 模型在不同推理引擎下数值行为一致。

8. **MATLAB 在线窗口维护**：在 MATLAB 中实现 128 步滑动窗口的在线维护，每个控制周期提取特征、归一化、更新窗口并按需推理。

9. **Simulink 接口封装**：通过 MATLAB Function 块薄包装将在线推理接入 Simulink 闭环模型，实现感知输出的实时调用。

10. **面向 LPV-MPC 调度验证的 theta 输出调理**：对坡度回归输出进行增益调整、绝对值限幅、变化率约束和死区处理，使感知输出适合闭环控制使用。

## 2. 项目事实边界

本软件当前主要用于桌面端 MATLAB/Simulink 闭环仿真与算法验证。ONNXRuntime 核心推理链路在项目测试中具备相对于 10 ms 控制周期的计算余量；MATLAB/Simulink 封装主要用于桌面仿真验证，不应表述为已经完成嵌入式硬实时部署。
