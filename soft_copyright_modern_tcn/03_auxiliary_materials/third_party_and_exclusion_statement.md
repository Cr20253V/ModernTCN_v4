# 第三方、数据与排除说明

## 1. 第三方依赖声明

以下第三方库作为运行环境或依赖工具，不作为本软件自有源代码：

1. **PyTorch**：深度学习框架，用于 ModernTCN 模型定义和训练；
2. **ONNXRuntime**：ONNX 模型推理引擎，用于一致性验证和实时性测试；
3. **NumPy/SciPy**：数值计算和数据加载；
4. **MATLAB/Simulink**：仿真平台和在线推理环境；
5. **onnx/onnxscript**：ONNX 导出辅助库。

## 2. 文件类型排除

以下文件类型不纳入源程序鉴别材料：

| 文件类型 | 说明 | 排除原因 |
|---|---|---|
| .pt | PyTorch checkpoint | 模型权重，非源程序 |
| .onnx | ONNX 模型 | 部署产物，非源程序 |
| .mat | MATLAB 数据文件 | 数据和中间结果，非源程序 |
| .slx | Simulink 模型 | 二进制工程文件，非源程序 |
| .slxc | Simulink 缓存 | 缓存文件，非源程序 |
| .png/.jpg | 图片 | 图表输出，非源程序 |
| .csv | 数据表 | 运行结果，非源程序 |
| .md | Markdown 文档 | 文档，非源程序 |
| .pdf | PDF 文档 | 文档，非源程序 |
| .docx | Word 文档 | 文档，非源程序 |

## 3. 自动生成代码处理

`src/ModernTCN/generated_layers/` 为 MATLAB 导入 ONNX 后自动生成的兼容层代码，默认不纳入本次核心源程序鉴别材料。若申请人决定纳入，应单独标注为"自动生成兼容层，不作为自有源程序主体"。

## 4. 排除目录

以下目录不纳入源程序鉴别材料：

| 目录 | 说明 |
|---|---|
| .git/ | 版本控制 |
| .github/ | 仓库平台配置 |
| .kilo/ | IDE 缓存 |
| .cursor/ | IDE 配置 |
| .venv/ | Python 虚拟环境 |
| __pycache__/ | Python 字节码缓存 |
| slprj/ | Simulink 自动生成缓存 |
| data/ | 训练数据和路径数据 |
| results/ | 运行结果和模型产物 |
| figures/ | 图表输出 |
| docs/ | 文档输出 |

## 5. 权属确认事项

申请人应最终确认以下事项：

1. 是否存在职务开发、合作开发、委托开发或第三方开源代码；
2. GitHub 公开仓库若构成首次发表，首次发表日期需由申请人确认；
3. 仓库中是否包含他人开源代码或单位职务成果；
4. 未把身份证、营业执照、公章、签名扫描件等敏感材料写入仓库。
