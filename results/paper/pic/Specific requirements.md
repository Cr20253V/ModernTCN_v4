# Fig. 3 Scheduling mismatch mechanism

## 作用

解释论文核心问题：LPV-MPC 的闭环性能依赖调度变量，尤其是坡度调度量。当 (\theta^{sch}_k \neq \theta^{true}_k) 时，会造成预测模型失配。

## 图像类型

机制示意图。
建议单栏或双栏。若想表达完整，可双栏。

```text
Preferred width: 3.5 in or 7.16 in
Format: PDF/SVG
```

## 推荐结构

中心放置：

```text
LPV-MPC prediction model
A(ρk), B(ρk), E(ρk)
ρk = [vk, ωk, θk^sch]
```

左侧放置四种调度来源：

```text
zero-slope: θk^sch = 0
IMU-based: θk^sch = θk^imu
oracle: θk^sch = θk^true
learned: θk^sch = S(θ_hat_k)
```

右侧放置影响链条：

```text
scheduling error eθ,k
→ model mismatch
→ inaccurate prediction
→ degraded tracking / aggressive control
```

底部可放对比逻辑：

```text
true slope: reference scheduling condition
zero/IMU/learned/oracle: compared scheduling sources
```

## 视觉要求

* true/oracle 路径用红色或黑色；
* learned 用蓝色；
* zero/IMU 用灰色/绿色；
* 用箭头表示信息流；
* 不要放真实数值。

## 不要出现

* 不要声称 learned slope 等于 true slope；
* 不要写 “guaranteed stability”；
* 不要放实验曲线，这张图是机制图，不是结果图。

## 建议 caption

```latex
\caption{LPV-MPC scheduling mismatch mechanism. Different slope scheduling sources provide different $\theta_k^{sch}$ signals to the same LPV-MPC prediction model. When $\theta_k^{sch}$ deviates from the true road grade $\theta_k^{true}$, the local prediction model becomes inconsistent with the nonlinear plant, which may lead to biased prediction, tracking degradation, and increased control variation.}
---
