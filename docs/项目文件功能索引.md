# LPV-MPC AGV 项目文件功能索引

## 项目概述
本项目是一个 **AGV 自适应 LPV-MPC 控制系统**，整合了：
- 非线性 AGV 动力学模型（S-Function）
- 线性参数变化（LPV）模型
- 模型预测控制（MPC）
- 工况识别（GRU 神经网络）
- 贝叶斯优化（参数调优）

## 📁 脚本文件分类

### 核心动力学模型（S-Function层）
| 脚本 | 功能 | 输入 | 输出 | 单位 |
|-----|------|------|------|------|
| **agv_model_sfunc.m** | Simulink Level-2 S-Function 包装 | u=[F_cmd; ω_cmd; θ_ground] | y (31×1) | N, rad/s, rad |
| **state_eq.m** | 非线性状态转移方程 | x, u, θ_ground | x_next (8×1) | - |
| **state_eq_ref.m** | 参考几何版状态转移 | x, u, θ_ground (读ref) | x_next (8×1) | - |
| **output_eq.m** | 输出方程（测量/观测/诊断） | x, u, θ_ground | y (31×1) | - |
| **output_eq_ref.m** | 参考几何版输出方程 | x, u, θ_ground (读ref) | y (31×1) | - |
| **parameters.m** | 集中参数定义 | - | params 结构体 | kg, m, m/s, rad |

**关键约定**：
- 状态向量 x (8×1)：[X, Y, ψ, v, ω, δ_lf, δ_rr, β]
- 输入向量 u (2×1)：[F_cmd, ω_cmd]
- 扰动 md：θ_ground（地面坡度角）
- 调度变量 ρ (3×1)：[v, ω, θ]（ω保留符号）
- 采样周期 Ts = 0.01 s

---

### 线性化与LPV模块
| 脚本 | 功能 | 输入 | 输出 | 备注 |
|-----|------|------|------|------|
| **lin_agv_at_point.m** | 单点线性化（有限差分法） | x0, u0, θ0 | A, B, C, D, E (离散) | 内部核心 |
| **lin_agv_grid.m** | 网格线性化 | grid {V, W, T}, params, opts | db：多维数组 A/B/C/D/E | 关键：输出离散LPV模型 |

**关键特性**：
- 离散化方法：ZOH 或 FOH
- 网格点建议：3×3×3 至 5×5×5
- 坐标系：路径坐标系误差 (e_y, e_ψ, e_v, e_ω)

---

### MPC 控制器设计
| 脚本 | 功能 | 输入 | 输出 | 备注 |
|-----|------|------|------|------|
| **mpc_setup_single_interp.m** | 创建自适应MPC | db (LPV数据库), opts (设计选项) | ctrl 结构体 | 关键：创建MPC对象和映射表 |
| **mpc_update_from_rho.m** | 在线模型/权重插值 | ρ=[v,ω,θ], db, maps | upd：A,B,C,D,E,Q,R,dR | 关键：在线参数更新 |

**关键参数**：
- 预测域 Np：1.5~3.0 s
- 控制域 Nc：0.5~1.0 s
- 输出权重 Q：[e_y, e_ψ, e_v, e_ω]
- 输入权重 R：[F_cmd, ω_cmd]
- MD 通道：θ（坡度角前馈）

## 🔌 核心接口说明

### 1. 调度变量 ρ（最重要！）
```matlab
ρ = [v;           % 纵向速度 [m/s]
     omega;       % 偏航角速度 [rad/s]（有符号，左转+，右转-）
     theta]       % 地面坡度角 [rad]（或 GRU估计 theta_hat）
```
**关键**：ω 保留符号，不取绝对值！

### 2. 线性化模型结构 db
```matlab
db.A      % [Nv×Nw×Nt×4×4] 状态矩阵
db.B      % [Nv×Nw×Nt×4×2] 输入矩阵
db.C      % [Nv×Nw×Nt×4×4] 输出矩阵
db.D      % [Nv×Nw×Nt×4×2] 直通矩阵
db.E      % [Nv×Nw×Nt×4×1] 扰动矩阵（MD 通道）
db.grid   % 网格定义 {V, W, T}
db.Ts     % 采样周期
```

### 3. MPC 控制器结构 ctrl
```matlab
ctrl.mpcobj  % MATLAB MPC 对象
ctrl.db      % LPV 数据库
ctrl.opts    % 设计选项
ctrl.maps    % 权重/约束映射表
  .Q_range, .R_range, .dR_range
  .omega_threshold, .q_y_gain_max
  .theta_threshold, .q_v_gain_max
  .R_F_gain_max_uphill, .R_F_gain_max_downhill
  ...
```

### 4. 更新结构 upd = mpc_update_from_rho(ρ, db, maps)
```matlab
upd.A, upd.B, upd.C, upd.D, upd.E  % 插值矩阵
upd.Q, upd.R, upd.dR               % 自适应权重
upd.umin, upd.umax                 % 动态约束
upd.rho_n                          % 归一化调度变量
```

### 5. GRU 状态分类器输出
```matlab
[state, out] = GRU_state_classifier('init', params, model)  % 初始化
[state, out] = GRU_state_classifier('update', state, y_raw) % 更新
% out 包含：
%   .theta_hat        % [rad]
%   .label_main       % ∈ {1,2,3,4}
%   .label_turn       % ∈ {-1,0,1}
%   .conf_main        % [4×1]
```

## ✅ 常用参数一览

| 参数 | 默认值 | 单位 | 说明 |
|-----|--------|------|------|
| `Ts` | 0.01 | s | 采样周期 |
| `mass` | 100 | kg | 车辆质量 |
| `L` | 2.0 | m | 轴距 |
| `Iz` | 16.67 | kg·m² | 偏航惯量 |
| `v_eq` | 1.0 | m/s | 平衡速度 |
| `omega_threshold` | 0.15 | rad/s | 转弯判定阈值 |
| `theta_threshold` | 0.035 | rad | 坡度判定阈值（≈2°） |
| `dwell_main` | 0.20 | s | GRU 平地驻留时间 |
| `dwell_turn` | 0.40 | s | GRU 转弯驻留时间 |
| `tau_theta` | 0.15 | s | GRU 坡度滤波时间常数 |
| `rho_filter_tau` | 0.4 | s | 调度变量低通滤波时间常数 |
| `theta_deadzone` | 0.015 | rad | GRU 坡度死区（≈0.86°） |

## 🎯 快速开始路线图

### 场景 1：仅运行闭环仿真（已有模型）
```matlab
% 步骤
1. open_system('LPVMPC_AGV_simulink.slx')
2. % 运行仿真
3. test_closed_loop_performance()  % 评估结果
```

### 场景 2：重新线性化并设计MPC
```matlab
params = parameters();
grid = struct('V_grid',[0.8;1.0;1.2], 'W_grid',[-0.2;0;0.2], 'T_grid',[-0.2;0;0.2]);
db = lin_agv_grid(params, grid, struct('coord','path','keep_E',true));
ctrl = mpc_setup_single_interp(db, struct());
save('lin_agv_db.mat', 'db');
save('ctrl.mat', 'ctrl');
```

### 场景 3：重新生成GRU训练数据
```matlab
GRU_gen_train_data();       % 生成原始数据
GRU_prepare_dataset();      % 数据预处理
GRU_train();                % 训练模型
test_GRU_workflow();        % 验证
```

### 场景 4：贝叶斯优化调参
```matlab
start_bayesian();           % 启动优化
% 等待完成，生成 maps_best.mat
plot_bo_figures();          % 可视化结果
```
