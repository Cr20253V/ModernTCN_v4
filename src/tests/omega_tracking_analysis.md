## 根因分析

### 控制力矩与阻尼力矩的平衡

在 `state_eq_ref.m` 的 `continuous_dynamics_core_v2` 函数中（约 L378-381）：

```matlab
C_damping = 400.0;                % 横摆阻尼系数 [Nm/(rad/s)]
Mz_damping = -C_damping * omega;  % 阻尼力矩
Mz_total = Mz_total + Mz_damping;
```

同时，偏航控制增益在 L193-199：

```matlab
K_omega_p = 100.0;                          % 偏航控制增益
omega_dot_desired = K_omega_p * e_omega;    % 期望角加速度
Mz_needed = I_z * omega_dot_desired;        % 所需偏航力矩
```

### 数值分析

以 ω_cmd = 0.1 rad/s 为例，稳态时 ω ≈ 0.08 rad/s：

- **阻尼力矩**: `Mz_damping = -400 × 0.08 = -32 Nm`
- **控制力矩**: `e_omega = 0.1 - 0.08 = 0.02`，`Mz_needed = 16.67 × 100 × 0.02 = 33.3 Nm`

两者接近平衡，导致角速度无法继续增长。

### 结论

**横摆阻尼系数 `C_damping` 相对于 `K_omega_p` 过大**，导致控制力矩被阻尼消耗约 20-25%。

## 修改建议

### 方案 A：降低阻尼（建议优先）

将 `C_damping` 从 400 降低到 **200**，可将跟踪率提升至约 90%。

- **优点**：不改变控制器增益，保持原有设计意图。
- **风险**：直线稳定性可能轻微下降，需验证。

### 方案 B：提高控制增益

将 `K_omega_p` 从 100 提高到 **150-200**。

- **优点**：更快的响应。
- **风险**：可能引起超调或振荡。

### 方案 C：综合调整（最佳）

- `C_damping`: 400 → **250**
- `K_omega_p`: 100 → **120**
